# Digital Signage BMFR Kelas II Manado

Proyek ini adalah sistem papan tanda digital untuk Balai Monitor Frekuensi Radio Kelas II Manado. Ini memungkinkan administrator untuk mengelola dan menampilkan konten di beberapa layar.

## Fitur

- Manajemen konten terpadu untuk semua layar
- Manajemen terpisah untuk tampilan eksternal dan internal
- Dukungan konten gambar dan video
- Penjadwalan konten
- Ukuran gambar yang diunggah otomatis diubah
- Mode tampilan layar penuh

## Pengaturan

1.  Buat database MySQL dengan nama `digital_signage`.
2.  Impor file `database.sql` ke dalam database `digital_signage`.
3.  Konfigurasikan koneksi database di `config.php`.
4.  Akses aplikasi di browser web Anda.
